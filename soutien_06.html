Quelques consignes générales

  • faire les points dans l'ordre et ne passer au point suivant que lorsque le précédent est achevé et testé,
  • utiliser la console autant que nécessaire Pour bien observer le contenu et le type des différentes variables,
  • écrire du code structuré et correctement indenté,
  • écrire du code largement documenté !
  • l'ajout du code JavaScript est fait entre les balises <script>...<script> de ce fichiet html

Eléments de départ fournis quel que soit l'exercice :

  • ce fichier (soutien_xx.html) qui est un conteneur de l'exercice
  • un fichier de style CSS (style.css) déjà chargé via le header du document html
  • un fichier Javascript (tools.js), déjà chargé via le header du document html, contenant divers outils :
    • une fonction d'affichage formatté d'un tableau (Array) à la fin du document html courant,
    • une fonction d'affichage formatté d'un résultat à la fin du document html courant,
    • une fonction de génération aléatoire d'un nombre entier compris entre 2 valeurs,
    • une fonction de génération d'une série aléatoire de nombres entiers rangés dans un tableau de type Array,
    L'utilisation de ces outils n'est pas systématiquement nécessaire (cela dépend des exercices). Par contre, vous n'avez, en aucun cas, besoin :
    • de modifier le code de ces outils,
    • de copier le code des fonctions contenues dans tools.js dans votre propre code.
      Si vous utilisez les fonctions outils, vous ne pouvez que les "appeler".
Que la force soit avec vous !

Soutien #04 - Filtrer un tableau

  1. À l'appui sur le bouton 'Générer', créez, à l'aide de la fonction outil randomArrayOfIntFromInterval, un tableau de 20 nombres entre 0 et 100.
    Cela se fait en écrivant la fonction genererTableau appelée lors du click sur le bouton.
  2. À l'appui sur le bouton 'Filtrer', en compétant la fonction filtrerTableau :
    • Lancez le parcours du tableau créé au point précédent (choisir le type de boucle le plus adapté) et construisez un nouveau tableau ne contenant que les nombres supérieurs à 50,
    • Faites l'affichage du tableau résultat en utilisant la fonction displayArray proposée dans les outils.
